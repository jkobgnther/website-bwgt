# Bewegungskoordinator Website

Website für den Bewegungskoordinator Tempelhof-Schöneberg - Ein Next.js Projekt mit Tailwind CSS.

## Technologie-Stack

- **Next.js 14** - React Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **React 18** - UI Library
- **Heroicons** - Icon Library
- **Resend** - Email Service

## Entwicklung

### Installation

```bash
npm install
```

### Entwicklungsserver starten

```bash
npm run dev
```

Öffnen Sie [http://localhost:3000](http://localhost:3000) im Browser.

### Build für Produktion

```bash
npm run build
npm start
```

### Linting

```bash
npm run lint
```

## Seitenstruktur

- `/` - Startseite
- `/services` - Beratungen & Services
- `/blog` - Blog / News / Infoblätter
- `/kontakt` - Kontaktformulare
- `/projektidee` - Projektidee & Konzept
- `/netzwerk` - Netzwerk der Bewegungslotsen
- `/impressum` - Impressum
- `/datenschutz` - Datenschutzerklärung
- `/barrierefreiheit` - Erklärung zur Barrierefreiheit

## Komponenten

Die Website verwendet wiederverwendbare Komponenten:

- `Header` - Hauptnavigation
- `Footer` - Fußzeile
- `ContactBanner` - Kontakt-Banner
- `ExpandableFAQ` - Erweiterbare FAQ-Sektionen
- `BerlinMapInteractive` - Interaktive Berlin-Karte
- `InstagramFeed` - Instagram Feed Integration
- `StartPageHeader` - Header für die Startseite
